OWASP Security Hardening

Systematic security audit and remediation of a production feature.

Audited the meeting rooms feature against the OWASP Top 10 and fixed every finding: parameterized SQL to eliminate injection, mounted auth middleware on unprotected routes, added authorization checks, input validation on all POST endpoints, URL scheme whitelisting, DB transactions to prevent vote-count race conditions, and HTML escaping to block XSS.

Key Decision

Wrapped the vote endpoint in a database transaction to prevent count drift under concurrent requests — a race condition that wouldn't show up in manual testing but could silently corrupt data under real traffic.

Horrible Goose

An admin-triggered chaos event with AI crowd-seeking, item theft, and honk mechanics.

Built an autonomous event creature with a multi-behavior state machine: the goose finds the densest player cluster, flies to it, then cycles through prowling, darting, item stealing, honking, and — yes — pooping. Users scatter in fear. Admin-only controls for release and recall.
